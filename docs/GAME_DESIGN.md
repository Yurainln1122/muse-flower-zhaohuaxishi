# 《暮色拾花》游戏设计

## 1. 一页体验定义

| 项目 | 定义 |
|---|---|
| 玩家是谁 | 原创“拾花校勘员”，受托在日落前整理一册会变形的旧事 |
| 主类型 | 空间化叙事推理益智 |
| 支持类型 | 轻资源管理；不与主循环平分重心 |
| 核心动作 | 翻看片段两面，必要时查证，用“留温 / 留刺 / 存疑”盖签，并在新证出现后改签 |
| 世界响应 | 正确处理会让纸园长出花、显出墨刺或稳定成留白；错误处理会出现“抵牾”并留下裂口 |
| 目标感受 | 亲手发现温情、批判与不确定可以共存；记忆的诚实来自愿意修订，不来自把一切判死 |
| 时长 | 首次完整游玩 12-18 分钟；熟练重玩 6-9 分钟 |
| 首发语言 | 简体中文；所有玩家可见文案集中在数据层，可替换 |

优先级：**校勘判断 > 世界反馈 > 资源压力 > 收集与包装**。发生范围冲突时，先删外围演出，再保护“新证迫使改签”。

## 2. 玩家承诺与体验支柱

一句话承诺：**你不是替过去下判决，而是在夕光中决定怎样诚实地保存它。**

| 支柱 | 最小可玩证据 |
|---|---|
| 先看背面，再相信标签 | 教学区“隐鼠”先给出猫的归因；完成阿长片段后出现反证，玩家必须重开旧片并改签 |
| 温情与芒刺同时成立 | 阿长与《山海经》是留温，但对隐鼠之死仍需存疑；两张卡以同一根线相连而不合并评价 |
| 不知道也是动作 | 范爱农水面没有隐藏“真凶”；只有存疑能让双重倒影稳定 |

## 3. 对标原则如何落地

- **整理改变路径**：每处三张片段处理完成后，册页折起成纸桥，打开下一处；不是菜单解锁。
- **物件承担叙事**：先看竹筛、绘图本、药引、红笔、白旗与水面关系，文字只补关键因果。
- **记忆成为规则**：新证会回流到旧卡，改变其可接受处理；没有“看完即消失”的剧情卡。
- **纸面即世界**：翻页、折痕、套色、红批、留白都同时承担视觉与状态反馈。

不借其他作品的角色、地图、UI、数值、关卡或文化符号。

## 4. 两层核心循环

### 5-30 秒短循环

`观察物件和题签 → 翻面读后果 → 决定是否花夕光查证 → 盖签 → 看到纸园响应 → 必要时改签`

- 普通玩家依靠两面短句、物件轮廓和即时抵牾提示。
- 熟练玩家会识别“表面美德 / 具体伤害”“权威纠正 / 事实关怀”“明确未知”等结构，减少查证和误签。

### 1-5 分钟组合循环

`处理一处三片 → 新证沿线回流 → 返回旧片改签 → 纸桥生成 → 进入更少提示的新处`

- 第一处公开签的含义并强制一次改签。
- 第二处撤掉颜色暗示，要求从具体后果判断。
- 第三处把相似外表的“红笔纠正”和“匿名指控”并置，检验玩家是否读因果而非读情绪。
- 第四处不给唯一事实补丁，检验玩家能否主动存疑。

## 5. 类型契约

| 契约 | 规则 |
|---|---|
| 可见信息 | 当前片段正面、已揭示背面、三枚签、剩余夕光、跨片段连线、即时世界反馈 |
| 隐藏信息 | 未查证的补证与尚未被其他片段触发的回流证据 |
| 提交点 | 盖签立即改变局部世界，但可在最终装帧前改签；“装帧”是阶段提交 |
| 随机性 | 无。固定片段与固定因果，保证同状态 + 同输入 = 同结果 |
| 夕光 | 共 6 格；只有主动“查证”与首次误签各消耗 1 格；翻面、改签和阅读不耗时 |
| 夕光耗尽 | 不判死局；进入“暮色校勘”，仍可完成，但不再显示额外查证提示，结果页标记为“凭现有证据成册” |
| 正确反馈 | 留温：花瓣展开；留刺：墨刺钉住虚假题签；存疑：双重轮廓稳定并保留空白 |
| 错误反馈 | 纸面裂口、短促木鱼/算盘错拍、“此签压不住抵牾”提示；允许立刻改签 |
| 完整结果 | 初次装帧得到“清醒成册”或“偏色之册”；后者可返回改签直至清醒成册 |
| 重开 | 结果页和暂停页均提供“重新拾花”，清空状态回到开场 |

## 6. 核心系统：三签校勘

三枚签判断的是**片段应该怎样被保存**，不是给人物打分。

| 签 | 适用因果 | 错用信号 |
|---|---|---|
| 留温 | 具体行动使关系、好奇或尊严得到保存 | 只有漂亮标签，没有可观察的人情行动 |
| 留刺 | 规范、权威或仪式的表面正当掩盖了具体伤害 | 只是悲伤或困难，没有可追责的外部机制 |
| 存疑 | 原作自我修正、证据冲突或明确不知道 | 已有清楚行动后果，却用“复杂”逃避判断 |

### 支持系统一：夕光查证

- 查证揭示一句更强证据或一条跨片连线。
- 6 格足以查一半片段；玩家需要判断哪里真的不确定。
- 查证不提供百科知识，只暴露作品内部的因果、反证或边界。

### 支持系统二：回流连线

- 片段之间以纸线相连；完成一片可能改变另一片的背面。
- 只使用四组必要联结：隐鼠↔阿长、孝图↔后记纠错、红笔讲义↔匿名信、白旗↔范爱农。
- 连线不会生成额外收集品，只用于迫使玩家重新观察。

## 7. 十二片段与正确处理

片段按玩法关系重组，不按原作篇目顺序排列。

| 场所 | 片段 | 正面 | 背面 / 关键证据 | 处理 | 回流 |
|---|---|---|---|---|---|
| 园墙根 | 隐鼠的空笼 | “猫吃了它。” | 后来得知并非猫所害；旧愤怒建立在误归因上 | 存疑 | 完成“绘图本”后回流 |
| 园墙根 | 绘图《山海经》 | 不识字的阿长问起这本书 | 她做成了别人不肯或不能做的事 | 留温 | 解开隐鼠反证 |
| 园墙根 | 雪地竹筛 | 拉绳越快越容易抓到鸟 | 关键是等鸟走到中间；耐心改变结果 | 留温 | 无 |
| 纸影街 | 郭巨的金釜 | “孝感神明，得金而归” | 儿童先看见的是自己可能被埋的恐惧 | 留刺 | 与“麻祜批注”相连 |
| 纸影街 | 船上的《鉴略》 | 背完书就能去看会 | 任务完成了，赛会与点心却失去意义 | 留刺 | 无 |
| 纸影街 | 活无常的算盘 | 勾魂使者理应可怕 | 民间偏爱他的爽直与人情，现实公正反而缺席 | 留温 | 无 |
| 药铺廊 | 经霜甘蔗 | 越难找的药引越灵验 | 稀有制造权威感，却未让病情好转 | 留刺 | 无 |
| 药铺廊 | 临终的喊声 | 礼俗说应大声唤回亲人 | 喊声让将平静的脸显出痛苦，留下终生愧疚 | 留刺 | 无 |
| 讲义桥 | 红笔血管图 | 先生改了学生的图 | “好看”不能移动事实；逐页订正是具体关怀 | 留温 | 与匿名信对照 |
| 讲义桥 | 匿名信 | 高分一定来自被泄题 | 弱国学生被预设为低能，指控没有证据 | 留刺 | 完成红笔后显出对照线 |
| 菱荡书案 | 白旗与布袍 | 光复后处处换了新旗 | 旧乡绅换上新名号，结构和处境仍旧 | 留刺 | 与范爱农相连 |
| 菱荡书案 | 双重水影 | 范爱农是失足还是自杀 | 叙述者明确写“不明白”，没有可补出的真相 | 存疑 | 最终检验 |
| 夕照书案 | 麻祜批注 | “麻胡子”被当作胡人 | 后记主动改正字源，并继续保留版本差异 | 存疑 | 作为装帧前最后一片；不计额外片数 |

实际游玩处理 12 张主片：夕照书案的“麻祜批注”替换掉已完成场所中最早的教学说明卡，因此界面始终不同时堆满 13 张。

## 8. 关卡节拍（15 分钟）

| 时间 | 阶段 | 玩家已知 | 新压力 | 可观察反馈 |
|---|---|---|---|---|
| 0:00-0:45 | 开场 | 受托在日落前整理旧册 | 只给三枚签，不解释原作背景 | 水横枝长出第一片叶；点击即进入，不播放长过场 |
| 0:45-3:30 | 教会：园墙根 | 翻面、盖签、改签 | “隐鼠”故意先给不完整证据 | 阿长片完成后纸线回流，旧卡轻响并出现双重脚印 |
| 3:30-6:30 | 变式：纸影街 | 三签因果已学会 | 去掉色彩暗示；表面赞词和具体后果相反 | 孝图翻面后金色题签裂成黑刺；纸桥从裂口长出 |
| 6:30-10:00 | 组合：药铺—讲义桥 | 查证有限，错误可改 | 两种“权威纠正”外观相似；必须区分事实关怀与无证指控 | 红笔让血管线归位；匿名信让字迹污开；药柜影子拉长 |
| 10:00-12:30 | 检验：菱荡书案 | 联结会回流 | “革命成功”与个体处境冲突；范爱农没有隐藏答案 | 白旗套色剥落，水面保留两道互不合并的倒影 |
| 12:30-15:00 | 装帧与结果 | 可自由回看所有片段 | 必须同时使用三签；不能把“存疑”当漏题 | 册页按玩家处理长成花、刺与留白；给出可操作的修订提示 |

首个有意义动作在 15 秒内出现。

## 9. 空间、镜头与路线

- 主镜头是略带俯角的横向纸卷，当前场所占视口 65%，书案工具占 35%。
- 桌面视口优先 `1366×768` 与 `1440×900`；窄屏改为上下布局，工具栏固定底部。
- 每处只有一个主路线和一条可选查证支线；不做自由寻路。
- 地标分别是皂荚树/竹筛、纸偶高帽、药柜/红笔桥、菱荡空船。
- 片段以可点击物件进入，不使用漂浮问号或小地图。

## 10. 反馈、失败与恢复

### 反馈层级

1. 输入：按钮压印、纸屑与短声。
2. 接触：签落在卡面，出现对应轮廓形状。
3. 因果：场景花/刺/雾改变，跨片纸线移动。
4. 阶段：纸桥折起并把镜头带向下一处。
5. 结果：整册装帧画面汇总玩家实际选择。

### 可理解失败

- **片段抵牾**：当前签与已揭示因果冲突；提示指出“哪条证据没被容纳”，不直接报标准答案。
- **偏色之册**：装帧时未使用三类签、或仍有至少三片抵牾；结果说明册页偏成“只怀旧 / 只控诉 / 处处含混”之一。
- **恢复**：从结果返回最后书案，所有片段状态保留；修订后可再次装帧。
- **重开**：清空状态，回到水横枝开场。

## 11. 最小试玩问题

最大设计假设：玩家能理解“签处理的是记忆呈现，不是人物善恶”。

最小试玩段：园墙根三片（隐鼠、绘图本、竹筛）。

观察问题：

1. 玩家第一次是否会翻面，而不是直接凭标题盖签？
2. 阿长片回流后，玩家是否主动回到隐鼠片？
3. 玩家能否解释为何“阿长可留温、隐鼠仍存疑”不矛盾？
4. 玩家是否把竹筛误当成知识问答，还是通过等待动画理解因果？

触发修改：若多数玩家只等文字公布答案，则减少解释句，增加物件动作和先后状态；若把签当人物评价，则在按钮文案中持续使用“怎样保存这段记忆”，并让同一人物连接不同签。

## 12. 范围与验收

### 必须包含

- 4 个连续场所、12 张主片、三签、夕光、四组回流连线。
- 可到达的“偏色之册”与“清醒成册”。
- 结果后返回修订与完全重开。
- 桌面和手机可操作；鼠标、触控与键盘均可完成。
- 简体中文无缺字、截断和关键术语漂移。

### 明确排除

- 原作全文、逐篇章节导航、人物扮演、战斗、开放世界、账号、存档云同步、排行榜、多语言、配音与生产级资产管线。

### 最小游玩验证

1. 页面加载后 15 秒内可翻第一片。
2. 阿长片能触发隐鼠回流，且改签状态持久。
3. 夕光只在查证与首次误签时减少，不会造成不可完成死局。
4. 三类签都能改变场景，并可在结果册页中辨认。
5. 偏色结果可返回修订；清醒成册可到达。
6. 重开恢复 6 格夕光、零片段状态和开场场所。
7. `1366×768`、`390×844` 无关键遮挡，中文按钮可操作。

## 13. 人工试玩边界

自动与代理验证只能证明运行、输入、规则、画面变化、结果和重开。是否真正“好玩”、节奏是否适合所有读者、文化表达是否足够细腻，需要中文玩家与文学读者实际试玩。
