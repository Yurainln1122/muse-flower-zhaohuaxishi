# 《朝花夕拾》游戏化来源圣经

## 1. 来源范围与边界

《朝花夕拾》不是线性小说，而是鲁迅在 1926 年写成、后来结集的十篇回忆散文。改编对象是整部散文集的共同结构：成年人回望童年与青年，在温情、讽刺、哀悼和自我校正之间重新安排记忆。

| 项目 | 结论 |
|---|---|
| 原文语言 | 中文；基础电子文本为繁体中文 |
| 策划语言 / 首发界面 | 简体中文 |
| 目标玩家市场 | 熟悉或想接近中国现代文学的中文玩家；不要求预先读过原作 |
| 基础版本 | Project Gutenberg eBook #25271，2021-01-03 更新，公版文本 |
| 补齐版本 | 维基文库《从百草园到三味书屋》《父亲的病》公版页面 |
| 授权边界 | 原作已进入公有领域；游戏仍只保留必要短引文，主体文案与玩家身份原创 |
| 边界真值 | 小引、十篇正文、后记，共 12 单元 |
| 覆盖结论 | 12/12；基础文本缺两篇，已补齐；无未处理失败 |

## 2. 全书覆盖表

| 单元 | 客观变化与回收 | 对可玩性的意义 |
|---|---|---|
| 小引 | 当下的纷扰触发旧事；作者明说记忆与实际可能有差异，旧味可能“哄骗”人反顾 | 记忆不是待复原的唯一真相；玩家应在保存、质疑与留白之间作决定 |
| 狗·猫·鼠 | 对猫的仇恨被归因于隐鼠之死，后来得知隐鼠其实被阿长踏死 | 同一碎片会被情绪错误归因；“先判断、后得新证”可形成可修正的系统 |
| 阿长与山海经 | 对阿长的厌烦、敬畏与怨恨，因她主动买来绘图《山海经》而重新排列 | 人不能被单一标签封存；物件能携带比身份偏见更强的关系证据 |
| 二十四孝图 | 儿童从图像接受规范，却在“老莱娱亲”“郭巨埋儿”中感到虚假与恐惧；后记又补证版本差异 | 图像不是中性装饰；玩家要识别“被包装成美德的伤害” |
| 五猖会 | 看赛会前被要求背《鉴略》，背出后虽获准出发，兴奋已被耗尽 | 正确完成任务也可能造成情绪失败；制度可改变奖励的意义 |
| 无常 | 民间舞台上的活无常因爽直、有人情而可亲，现实中的“正人君子”却缺乏公正 | 规则与人情不是简单二选一；角色的可信度可由行为而非名号建立 |
| 从百草园到三味书屋 | 百草园的探索、传说与捕鸟需要观察和耐心；进入书塾后问题受限，但孩子仍从小园、画绣像中寻找缝隙 | 好奇心会迁移而非消失；同一核心动词可在自由空间和受限空间产生变式 |
| 父亲的病 | 家庭反复寻找昂贵、难得的药引；病情仍恶化；临终礼俗让父亲更痛苦，叙述者留下长期愧疚 | “更稀有/更用力”不等于更有效；照料需要读懂反馈并承认不可挽回 |
| 琐记 | 衍太太表面纵容孩子、背后散布闲话；叙述者离乡求学，从水师学堂转矿路学堂并接触《天演论》 | 表面友善与真实后果可分离；新知识提供逃离旧路径的工具，也暴露新制度的荒谬 |
| 藤野先生 | 藤野认真改讲义、纠解剖图；匿名信与幻灯片事件改变“我”的道路，最终弃医从文 | 校勘是关怀，也是对事实形状的尊重；被纠正不等于被羞辱 |
| 范爱农 | 初见互相误解，后成朋友；革命后旧人换装、结构未变；范爱农失业漂泊，死因仍不确定 | 玩家不能用一个胜利标签覆盖系统是否改变；重要结局可以保持“存疑” |
| 后记 | 作者主动纠正“麻胡/麻祜”，比较孝图和无常图像的版本差异，也承认判断仍有限 | 最终结算必须允许修订与注脚；承认不知道是有效而非失败的动作 |

覆盖集合与来源边界集合一致：`{小引, 狗·猫·鼠, 阿长与山海经, 二十四孝图, 五猖会, 无常, 从百草园到三味书屋, 父亲的病, 琐记, 藤野先生, 范爱农, 后记}`。

## 3. 世界身份

这部作品的独特世界不是“鲁迅故居观光”，而是三层同时存在：

1. **被记住的日常**：园子、书塾、药铺、赛会、学堂、宿舍、酒馆和码头。
2. **覆盖其上的解释**：家规、孝道图、名医话术、革命口号、留学生偏见、民间鬼神。
3. **多年后的校勘**：温情会放大味道，芒刺会暴露虚假，新证据会迫使旧判断改写。

游戏空间可以将这些地点压缩成一座“夕照记忆园”，但不得声称它是原作真实地理。

## 5. 玩家身份候选

| 身份 | 可访问动作 | 风险 |
|---|---|---|
| 原创“记忆校勘员” | 拾取、翻面、查证、归档、留白、改签 | 最能避开复演；需避免变成纯阅读界面 |
| 原创赛会扎作学徒 | 制作纸偶、安排队列、辨认民间/书本形象 | 视觉强，但容易过度集中《无常》《五猖会》 |
| 原创离乡学生 | 在园、学堂、药铺和码头选择携带物 | 情绪自然，但容易被误读成代演鲁迅成长线 |

后续概念优先验证“记忆校勘员”。其身份依据来自全书反复出现的收集绘图、改讲义、比较版本、修正记忆和保存人物痕迹，但角色本身完全原创。

## 6. 可玩动词

- **拾取**：从空间中选择有限数量的物件或片段。
- **翻面**：让温情表象、制度标签或传说显出另一层后果。
- **查证**：消耗时间/机会，降低片段的疑雾，暴露错归因或版本差异。
- **归档**：用“留温、留刺、存疑”三种签将片段放入记忆册。
- **改签**：新证据出现后，承认旧判断不够完整并修改。
- **等待**：少数情境中不立即操作，观察世界自己显露规律。
- **联结**：把跨篇重复的物件、人物或制度反应连成因果。

这些动作都允许选择、即时反馈和熟练度差异；单纯点“下一段”不算核心动词。

## 7. 世界模型

最小因果：

`片段当前面貌 + 玩家校勘动作 + 证据规则 → 新的温度/芒刺/疑雾 + 空间、声音与路线反馈`

| 状态 | 含义 | 可见反馈 |
|---|---|---|
| 温度 | 具体的人情、好奇、照料或纪念被保留 | 暖色套印、花瓣展开、环境音变近 |
| 芒刺 | 虚假礼法、权威话术或制度伤害被看见 | 墨刺、刻线显形、背景纸面裂开 |
| 疑雾 | 证据冲突、记忆偏差或原作明确不知道 | 半透明留白、双重轮廓、问号形印章 |
| 夕光 | 本局剩余行动机会，而非急促倒计时 | 影子变长；允许思考，但查证不能无限使用 |

优秀玩家不会把所有片段都盖成同一种签，而会在发现新证据后改签，并让最终册页同时容纳温情、批判与不确定性。

## 8. 空间、通路与角色能动性

| 空间 | 地标与门槛 | 世界如何回应 |
|---|---|---|
| 园墙根 | 石井栏、皂荚树、断砖、捕鸟竹筛 | 急于拉绳只得空筛；等待显示隐蔽轨迹 |
| 纸影街 | 孝图、赛会纸偶、活无常高帽与算盘 | 图像翻面后，赞词可能变成受害者视角 |
| 药铺廊 | 高柜台、药引抽屉、当票、喘息灯 | 追逐稀有药引会耗尽夕光，却不保证病情改善 |
| 书塾—讲义桥 | 《鉴略》、绣像、红笔讲义、匿名信 | 同为“纠正”，反馈区分关怀校勘与权力压迫 |
| 菱荡码头 | 白旗、布袍、酒盏、空船 | 胜利标志与人物处境并置；范爱农结局只能存疑 |
| 夕照书案 | 水横枝、旧册、三枚签 | 汇总玩家保留的关系，而非给原作人物判分 |

原作人物只以记忆回声、物件关系和短句出现；他们有自己的行为依据，不成为等待玩家“修复”的任务 NPC。

## 9. 情绪节奏

1. **好奇**：玩家先在园中发现可翻面的物件，动作轻快。
2. **迟疑**：同一物件的表面标签与实际后果冲突。
3. **压迫**：赛会、药铺、匿名信等让“完成”与“正确”失去简单意义。
4. **人情回光**：阿长的书、藤野的红笔、活无常的爽直证明温情不等于粉饰。
5. **不可裁决**：范爱农之死和记忆差异要求玩家主动留下空白。
6. **带刺成册**：最后结果不是完美真相，而是一册诚实、仍可修订的记忆。

## 10. 文化语境与术语

- 清末民初绍兴民俗、书塾教育、孝道图像、迎神赛会、传统医药话术、留学与革命语境必须保持差异，不能统一包装成“东方神秘学”。
- “无常”首先是越地民间戏会中带有人情的勾魂角色，不做通用恐怖死神。
- “孝”在作品中不是可简单嘲笑的词；批判对象是用惨苦、虚假和恐惧规训儿童的图像与话术。
- 阿长、藤野先生、范爱农都不可简化成善/恶卡牌。
- 首发界面为简体中文；原文短引保留原字形时附简体释义，不要求玩家阅读繁体。

统一术语：

| 原作词 | 游戏用词 | 说明 |
|---|---|---|
| 旧事重提 | 旧事 / 记忆片段 | 不改作“回忆副本” |
| 活无常 | 活无常 | 不译为死神 |
| 五猖会 | 五猖会 | 作为赛会名称，不泛称庙会 |
| 《鉴略》 | 《鉴略》 | 强记材料与被抽走的快乐相连 |
| 《山海经》 | 绘图《山海经》 | 强调阿长买来的有图版本 |
| 存疑 | 存疑签 | 不是“错误”或“未完成” |

## 11. 视觉与声音锚点

- 夕阳穿过西窗，水横枝浸在玻璃瓶中。
- 百草园墙根：断砖、何首乌藤、覆盆子、竹筛与长绳。
- 粗木刻/石印的套色错位，红批落在黑字旁。
- 活无常的白衣高帽、纸扇与算盘；必须保留民间戏台的亲近感。
- 五猖会的船、吹打声与被《鉴略》压低的儿童视线。
- 药铺高柜台、当铺更高的柜台、难寻药引和逐渐拉长的喘息。
- 藤野红笔修正的血管图，与匿名信的污黑字迹形成对照。
- 菱荡水面、空船、白旗和始终没有答案的双重倒影。

声音层级：虫鸣/翻纸/木刻摩擦为常态；查证时加入红笔、刻刀和纸张翻面的近声；制度压力用远处背诵、算盘、喘息与鼓点叠压；“存疑”后主动减声，保留空白。

## 12. 风险与开放设计空间

| 风险 | 处理 |
|---|---|
| 做成电子读书报告 | 核心结果必须由空间选择、有限行动与可修正状态产生 |
| 逐篇复演 | 地点与物件跨篇重组；玩家不扮演鲁迅或原作人物 |
| 怀旧滤镜抹平批判 | 最终册页必须同时检查温度、芒刺与疑雾 |
| 把传统文化视觉化为猎奇符号 | 每个民俗元素都绑定具体功能和语境 |
| 对父亲、阿长、范爱农作道德审判 | 只判断当前记忆处理是否诚实，不给人物打分 |
| 15 分钟内文字过多 | 每片段正文限 2-4 句；信息主要由位置、翻面和反馈表达 |
| “正确答案”过强 | 只对有明确证据的错归因判错；原作未定处以存疑为最佳动作 |

开放设计空间包括园中原创引路纸燕、玩家工具外形、空间压缩方式和关卡顺序；不得改写原作人物的关键行为或为范爱农之死给出确定答案。
