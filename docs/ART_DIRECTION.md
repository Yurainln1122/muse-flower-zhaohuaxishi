# 《暮色拾花》美术与创意方向

## 1. 核心视觉原则

1. **一切像刚从旧册里立起来**：场景、物件和界面共享纸张、折痕、木刻线与套色语法；排除写实 3D、通用国风山水和现代卡片仪表盘。
2. **状态先靠形状，再靠颜色**：留温是圆瓣与包裹线，留刺是尖刻线与裂口，存疑是双轮廓与留白；排除只靠红绿区分对错。
3. **旧物必须参与判断**：竹筛、绘图本、药引、红笔、白旗、水面都可操作且有因果；排除把民俗物件堆成背景装饰。
4. **人物只留不完整的痕迹**：使用手、袖口、背影、纸偶和物件关系，不制作鲁迅或原作人物的可消费肖像；排除名人传记插画感。
5. **夕照不是滤镜，是资源**：光线随查证和误签拉长，改变纸面可读性和影子方向；排除始终不变的怀旧橙色调。

反向原则：不使用龙纹、灯笼、祥云等无功能“古风符号包”；不把活无常画成西式死神；不把药铺、书塾、留学场景混成同一时代布景。

## 2. 镜头与构图

- 桌面：`1366×768` 基准。主纸卷位于左侧约 68%，校勘书案在右侧约 32%。
- 手机：`390×844` 基准。场景占上部 58%，片段抽屉与三签固定在下部；正文弹层不超过视口 70%。
- 俯角约 20° 的“纸上舞台”，不模拟真实透视；前景是可点物件，中景是纸路，后景是折起的墙与树。
- 焦点顺序：当前片段 → 三签决策 → 跨片连线 → 夕光。
- 允许装饰遮挡折角和边框，不允许遮挡片段标题、签按钮、夕光和返回路径。
- 动画镜头只做轻微横移与纸桥折叠；不晃屏、不快速缩放。

## 3. 世界与角色语法

### 形状

- 玩家：右下角一枚空心圆章和一只无身份手影；保持原创、无具体年龄性别。
- 可操作片段：竖向圆角纸签，左上有缺口，背面有明显折页角。
- 目标：圆瓣、闭合书脊和连续纸线。
- 威胁/抵牾：向内刺的刻线、断裂书脊、错位套色。
- 存疑：同一轮廓横向错开 4-8px，中间保持原纸色。

### 场所

- 园墙根：低密度草叶与虫线，高识别度竹筛和长绳。
- 纸影街：纵向纸偶与横向船板构成节奏；活无常白衣高帽但轮廓亲近、身体微向人群倾斜。
- 药铺—讲义桥：高柜台压迫上方，细红线跨过黑墨图；空间从窄暗向开阔纸桥过渡。
- 菱荡书案：大面积低密度水纹和空船，白旗只有少量套色，双重倒影是核心可玩信息。

## 4. 材质、光与颜色

### 纸与墨

- 底纸：暖灰宣纸 `#E9DFC8`，用 CSS 细噪点、纤维渐变和轻微边缘暗化表现，不使用大图纹理。
- 主墨：`#211B18`；次墨：`#574A40`。
- 红批：`#A33A2B`，只用于校勘、警示与重要连线。
- 留温套色：赭黄 `#C88A3D` + 圆瓣形。
- 留刺套色：黛青 `#31535A` + 尖刻线。
- 存疑套色：雾紫 `#777087` + 双轮廓/网点。
- 夕光：`#E4B765` 到 `#6E5961` 的方向性变化；每耗一格，阴影向左延长，背景饱和度降低一点。

所有状态同时有图形和文字标签，保证灰阶下仍可区分。

## 5. 动作、反馈与声音

| 事件 | 视觉 | 声音 |
|---|---|---|
| 翻面 | 纸片沿竖轴折转，背面套色稍错位 | 短促翻纸声 |
| 查证 | 红线从书案划到片段，显出补证 | 铅笔/红笔摩擦两拍 |
| 留温 | 圆瓣从签边展开，物件影子靠近 | 木鱼般温和单音 + 近处虫鸣 |
| 留刺 | 黑线刺穿虚假题签，表层剥落 | 刻刀划纸 + 低算盘错拍 |
| 存疑 | 双轮廓分开后稳定，中央留白 | 两个相近但不合一的纸铃音 |
| 错签 | 卡边向内塌、裂口短闪，不抖全屏 | 干脆“咔”声 |
| 回流 | 纸线穿过折页，旧卡角轻抬 | 远处风铃沿左右声道移动 |
| 清醒成册 | 花、刺、留白共同压入封面 | 虫鸣、翻页与一记鼓点收束 |

无音频素材时允许 Web Audio 生成极短反馈音；默认音量克制，提供静音按钮。

## 6. 界面与中文排版

- 标题：中文宋体/衬线字体栈（`Noto Serif SC`, `Source Han Serif SC`, `Songti SC`, `SimSun`, serif），字重 700。
- 正文与按钮：中文黑体栈（`Noto Sans SC`, `Microsoft YaHei`, `PingFang SC`, sans-serif）。
- 正文 16-18px，行高 1.75；移动端不小于 15px。
- 横排从左到右；原作短引使用缩进和细竖线，不模拟难读竖排。
- 单张片段正面不超过 32 个汉字；背面不超过 72 个汉字；查证补句不超过 48 个汉字。
- 三签按钮始终同时显示“名称 + 形状图标 + 一句动词释义”，不只显示颜色。
- 当前片段、剩余夕光、返回场所、静音和重开均可键盘聚焦；焦点环使用双线墨框。
- 不把文字烙进图片；首版只有简体中文。

## 7. 三张招牌游戏画面

### 初见：园墙根的第一道反证

- 玩家动作：正准备给“隐鼠的空笼”盖签。
- 目标与压力：六格夕光完整，但卡背仍锁着一行反证。
- 原作规则：空笼旁的猫影与阿长脚印出现错位，暗示最初归因不可靠。
- 第一焦点：空笼卡上方半透明的双重轮廓。
- 空间层次：前景竹筛和绳，中景空笼与绘图本，后景皂荚树折页。
- UI：右侧三签像实体印章排开；不显示教程大框。

### 核心动作：红笔与匿名信并置

- 玩家动作：把“留温”盖在红笔讲义上，红线令错位血管归位。
- 目标与压力：匿名信同时渗出污黑字，要求玩家判断同样是“纠正”，为何后果不同。
- 原作规则：事实不能因好看而挪动；无证指控建立在偏见上。
- 第一焦点：红笔线和黑污线在桥中央交叉但不混色。
- 空间层次：前景药柜抽屉，中景讲义桥，后景半透明课堂窗。
- UI：夕光只剩三格，查证按钮仍可用一次。

### 高潮/结果：菱荡上的清醒成册

- 玩家动作：给范爱农的双重水影盖“存疑”，随后装帧。
- 目标与压力：必须接受没有隐藏答案；错误强判会让两道倒影互相撕裂。
- 原作规则：光复的白旗与旧结构并存，范爱农死因仍不明。
- 第一焦点：两道互不合并但稳定的倒影；其上浮着一页留白。
- 空间层次：前景酒盏与空船，中景水影，后景夕阳书脊。
- UI：三签痕迹在册页边缘同时可见，结果按钮写“装帧这册旧事”。

## 8. 原创与授权边界

- 原作公版，只使用必要短引文和事实锚点；玩家、引导语、界面、关卡重组与所有图形均原创。
- 不使用鲁迅、阿长、藤野先生、范爱农的照片或现代影视形象。
- 不临摹《Pentiment》或任何现代游戏的角色、地图、UI 与资产；只借“纸面媒介服务玩法”的抽象原则。
- 若后续加入历史图片，必须逐张确认公版或可商用许可，并记录来源；首版不依赖外部图片。

## 9. 仍需验证的视觉风险

- 纸纹和套色不能降低中文小字可读性；移动端必须实机截图检查。
- 活无常的高帽、白衣和算盘要有越地民间戏会语境，不能显得像通用恐怖敌人。
- 三种状态在灰阶、色弱和低亮度下仍需靠形状成立。
- 片段弹层不能遮住世界反馈，否则玩家只会读卡而看不到因果。

交付判断：本方向能一眼区分于通用古风网页；玩家、目标与抵牾轮廓明确；核心规则可见；三张画面分别展示教学、组合和结果；界面只显示当前决策需要的信息。
