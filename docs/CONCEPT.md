# 《朝花夕拾》网页游戏概念

## 一页产品定义

| 项目 | 定义 |
|---|---|
| 暂定名 | **《暮色拾花》** |
| 玩家身份 | 原创角色“拾花校勘员”，第一次进入一座会随判断变形的记忆园 |
| 主类型 | 空间化叙事推理益智 |
| 支持类型 | 轻资源管理 |
| 视角 | 单屏展开的纸上园林；局部场景以横向纸卷切换 |
| 平台 / 输入 | 桌面与移动网页；鼠标、触控，键盘可完整操作 |
| 单次时长 | 12-18 分钟，目标 15 分钟 |
| 目标市场 | 中文玩家，尤其是中学生以上、文学与独立游戏受众；不要求读过原作 |
| 首发语言 | 简体中文 |
| 玩家幻想 | 在夕光耗尽前，把一册记忆整理得既有温度、又保留芒刺，并敢于留下不知道的空白 |
| 核心系统 | 翻看片段的两面，用“留温 / 留刺 / 存疑”处理；新证据可迫使玩家改签 |
| 非目标 | 不复演鲁迅生平；不做十篇章节关卡；不做战斗、开放世界、养成树、知识问答或全文阅读器 |
| 最大未知 | 三种签能否形成可学习的判断，而不是披着交互外衣的道德选择题 |

### 三条体验支柱

| 支柱 | 可观察试玩证据 | 否决现象 |
|---|---|---|
| **先看背面，再相信标签** | 玩家在首个片段主动翻面，并因新证据修改一次选择 | 玩家只按标题或颜色机械选“正确答案” |
| **温情与芒刺能同时成立** | 最终册页中至少有一组互相牵连、却被不同方式处理的片段 | 玩家认为“暖=好、刺=坏”，只追单一数值 |
| **不知道也是一种诚实动作** | 面对范爱农结局时，玩家理解“存疑”不是漏做或失败 | 玩家把所有片段都强行定性，或认为存疑没有反馈 |

## 行业对标矩阵

只借已被官方资料明确说明的单一设计原则，不复制角色、地图、界面、文本、美术资产或数值。

| 对标 | 官方事实 | 可迁移原则 | 本作落地 | 明确不借 |
|---|---|---|---|---|
| [The Gardens Between](https://thegardensbetween.com/) | 官方说明玩家操纵时间解谜，并在梦境般的花园中照亮苦乐参半的记忆 | 让“回忆”成为可操作的世界规则，而非只用旁白说明 | 翻面与改签会让同一园景显出另一层因果 | 不借双主角、岛屿关卡、时间回放机制与视觉资产 |
| [Carto](https://www.humblegames.com/games/carto/) | 官方说明玩家重排地图碎片，从而改变世界并打开路径 | 整理动作应直接改变可达空间 | 片段归档正确后，纸路折起并连到下一处；错误处理会形成绕路或雾障 | 不借地图拼块造型、角色、世界与谜题结构 |
| [Unpacking](https://www.unpackinggame.com/) | 官方说明通过摆放物件，在不直接讲述的情况下了解一个人的生命线索 | 物件位置与去留可以承担叙事 | 《山海经》、红笔讲义、药引、酒盏等通过关系位置说话，短文案只补必要因果 | 不借房间收纳、无形主角生平、像素资产与八次搬家结构 |
| [Pentiment](https://pentiment.obsidian.net/) | 官方说明其视觉受手抄本与早期木刻启发，选择会产生长期后果 | 书页媒介可以同时是世界、界面与记忆层 | 用原创的晚清木刻、石印套色与批注语法表现“可校勘的记忆” | 不借欧洲手抄本符号、巴伐利亚设定、角色、侦探故事与界面布局 |

文化适用性：上述作品证明的是“记忆可操作、空间可重排、物件可叙事、纸面可成为世界”这些玩法原则；《朝花夕拾》的绍兴民俗、孝道图像、药铺、书塾、留学与革命语境必须从中文原作重新建立。目标玩家不应靠欧洲宗教画、日式妖怪或通用古风去理解本作。

## 三个方向

### A. 《暮色拾花》：记忆校勘园

- **类型 / 视角**：空间化叙事推理益智；纸上园林单屏 + 局部横卷。
- **玩家身份 / 幻想**：原创“拾花校勘员”，替一册会自我改写的旧事做最后一次夜前校勘。
- **核心动词**：翻面、查证、盖签、改签、联结。
- **循环**：观察物件与环境 → 选择是否花夕光查证 → 盖“留温/留刺/存疑”签 → 园景改变并出现新证 → 必要时改签。
- **压力 / 技巧**：查证次数有限；熟练玩家会从轮廓、声音和跨片段关系判断哪里值得查，不按情绪表面作答。
- **原作转化**：把“旧味、讽刺、自我修正”变成同一套可重复动作；不要求玩家替原作人物做决定。
- **最相关原则**：Carto 的“整理即改变路径” + Unpacking 的“物件关系叙事”。
- **传播画面**：夕照书案中央是一册半展开的记忆园；左页开着《山海经》的奇兽木刻，右页红笔讲义与匿名信叠压，远处白衣无常沿纸桥走过。
- **原型证明**：三种签是否产生可学习、可修正的推理；四处空间、八个核心片段、一个最终装帧。
- **不做**：全文摘录、角色对话树、真实地理复原、十篇逐章关卡。
- **最大风险**：玩家把它当成“选主题词”的阅读题。
- **最小验证问题**：首组三个片段中，玩家能否仅凭翻面与世界反馈，理解“新证会改变旧判断”？

### B. 《纸马会》：赛会扎作学徒

- **类型 / 视角**：队列编排策略；侧视长卷。
- **玩家身份 / 幻想**：原创纸扎学徒，在赛会开始前安排纸偶、锣鼓与路障，让人情穿过僵硬礼法。
- **核心动词**：扎作、排序、引导、让路。
- **循环**：观察街坊需求 → 编排队列 → 放行赛会 → 看拥堵、恐惧与欢笑反馈 → 重排。
- **压力 / 技巧**：空间和节拍有限；单位既有功能也有文化含义。
- **原作转化**：集中《五猖会》《无常》《二十四孝图》的民间图像与制度讽刺。
- **最相关原则**：The Gardens Between 的“情绪通过空间谜题显现”。
- **传播画面**：套色纸偶队列穿过窄街，白衣活无常侧身给孩子让路，孝图木牌在雨中裂开。
- **原型证明**：单一队列规则能否同时承载人情与批判。
- **不做**：战斗、节庆经营、神怪收集。
- **最大风险**：虽有强画面，却把整部作品缩成三篇民俗散文。
- **最小验证问题**：玩家能否从队列后果理解“规则有人情才可亲”，而非只追最短通路？

### C. 《离港清单》：一只原创行李箱

- **类型 / 视角**：物件摆放与取舍；俯视箱格。
- **玩家身份 / 幻想**：原创离乡学生，在启程前从一屋旧物中只带走有限物件，并在途中发现它们的另一面。
- **核心动词**：拿取、摆放、舍弃、重看。
- **循环**：查看物件 → 装箱受空间限制 → 到站后触发关系后果 → 重新取舍。
- **压力 / 技巧**：箱格与重量有限；真正重要的物件未必价值高或形状好放。
- **原作转化**：用《山海经》、绣像、白袜、讲义、照片、酒盏串联求学与人物关系。
- **最相关原则**：Unpacking 的“物件摆放承载未直说的故事”。
- **传播画面**：打开的藤箱里，红笔讲义压着粗木刻《山海经》，一双无用白袜卡住箱角，水面倒影从箱底渗出。
- **原型证明**：有限取舍能否让玩家感到“记住什么，就是成为什么”。
- **不做**：搬家模拟、库存养成、旅行地图。
- **最大风险**：与 Unpacking 的表面形式过近，且容易变成一次性最优装箱题。
- **最小验证问题**：第二站出现新证后，玩家是否愿意主动拿掉高价值物，换回关系物？

## 比较与选择

### 硬否决检查

- A：有重复动作、压力、改签与世界反馈；不依赖复演或大型内容，**通过**。
- B：机制成立，但整部作品覆盖不足是硬范围风险，**淘汰**。
- C：机制成立，但与对标表面相似且全书批判层偏弱，**淘汰**。

### 关键取舍

- **原作适配**：A 唯一把“小引的记忆偏差”和“后记的主动校正”纳入核心循环，因此能统摄十篇。
- **玩家能动性**：A 允许玩家查证、误判、改签和留白；结果不是沿固定叙事点击。
- **视觉传播**：A 的纸上园林可在一张画面同时出现园墙根、木刻纸偶、红笔讲义和夕照书案。
- **文化适用**：A 不需把某一民俗奇观当作全书代表，也能保留绍兴物件的具体语境。
- **完成风险**：A 可用 8 个片段和 4 个空间证明核心；无需复杂寻路、物理或大量角色动画。

## Quick 选择状态

**已自动选择 A：《暮色拾花》：记忆校勘园。**

### 不可妥协项

1. 玩家必须是原创校勘员，不扮演鲁迅、童年叙述者或任何原作人物。
2. 核心动作必须包括“翻面/查证—盖签—因新证改签”，不能退化成知识选择题。
3. 最终好结果必须同时容纳温度、芒刺与至少一处存疑；不是全收集或全答对。
4. 原作人物不接受玩家道德评分；签处理的是记忆片段的呈现方式。
5. 15 分钟内有清楚开场、四阶段升级、至少一种可理解失败、结果与重开。

### 最小验证问题

只做“隐鼠—阿长—《山海经》”三个互相关联片段：玩家先可能将阿长盖成“芒刺”，获得新证后是否能理解并主动改签为“留温”，同时仍保留人物的复杂面？若玩家只是在等系统公布标准答案，就要把证据变成空间与资源决策，而不是追加解释文字。

### 开放问题

- 查证机会以固定次数、夕光格还是空间绕路表达，需以首轮可读性决定。
- 片段总数在 8-10 之间；优先保留跨篇联结，不追求每篇一枚。
- 是否提供“无文字挑战”不在首版范围；首版保证中文文案集中可替换。
