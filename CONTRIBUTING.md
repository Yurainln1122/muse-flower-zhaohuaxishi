# 参与贡献

感谢你愿意改进《暮色拾花》。

## 报告问题

提交 Issue 时，请说明：

- 使用的设备、浏览器和视口大小；
- 问题发生在哪一景、哪张残片；
- 可以稳定复现的操作步骤；
- 实际结果与预期结果；
- 如方便，请附截图或控制台错误。

## 提交代码

1. 从 `main` 创建功能分支。
2. 保持玩家原创身份、跨篇重组和三签判断系统不变。
3. 不要加入逐段复演原作的线性剧情。
4. UI 状态不能只依赖颜色，应同时保留文字或形状提示。
5. 提交前运行：

```bash
pnpm lint
pnpm test
```

Pull Request 请说明改动目的、玩家体验影响和验证方式。

## 内容原则

- 对历史人物和文化语境保持准确、克制。
- 不把存在证据空白的情节改写为确定事实。
- 新增文案应使用简体中文，并避免大段复制原文。
